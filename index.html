The 2-Day Battle Plan

Day 1 — The Foundations

6 hours
  • M01

    Welcome to the Forge

    Why Go? Go's philosophy, where it shines, and what makes it different from the crowd.

    30 min
  • M02

    Setup & Hello, Gopher!

    Install Go, configure your editor, write your first program and actually understand what it does.

    30 min
  • M03

    Variables, Types & Constants

    Go's type system, type inference with :=, zero values, and why Go doesn't let you be lazy about types.

    1 hour
  • M04

    Control Flow

    if, for (the only loop you'll ever need), switch — and why Go's for is secretly a superpower.

    1 hour
  • M05

    Functions & Error Handling

    Multiple return values, naked returns, and the infamous if err != nil — you'll learn to love it.

    1 hour
  • M06

    Arrays, Slices & Maps

    Go's core data structures. Slices are not arrays (this will save you). Maps are your best friends.

    1 hour
  • 🔨

    Project #1 — CLI Task Manager

    Build a fully working command-line todo app from scratch. Add, list, complete, delete tasks. Real code, real terminal.

    1 hour

Day 2 — The Real World

6 hours
  • M07

    Structs & Interfaces

    Go's answer to objects. Structs hold data, interfaces define behavior — together they're unstoppable.

    1 hour
  • M08

    Goroutines & Channels

    The magic of Go concurrency. Launch a thousand goroutines. Use channels to communicate. Don't panic (pun intended).

    1 hour
  • M09

    Packages & Modules

    go mod init, organizing code into packages, importing third-party libraries. Your code grows up.

    30 min
  • 🔨

    Project #2 — Concurrent Web Scraper

    Fetch multiple URLs simultaneously using goroutines. See the speed difference. This is why devs fall in love with Go.

    30 min
  • M10

    Building REST APIs with net/http

    No frameworks, just Go's stdlib. Create routes, handle JSON, respond to requests. Pure, clean, fast.

    1.5 hours
  • 🏆

    Capstone — URL Shortener API

    Combine everything: structs, maps, HTTP handlers, JSON, and a touch of concurrency. Ship a real working API by the end of Day 2.

    1.5 hours
