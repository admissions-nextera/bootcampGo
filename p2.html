Step 1 — Imports, ScrapeResult struct & URLs

Start with the scaffolding: the struct that captures everything we want to know about each URL's scrape result, and the list of URLs to scrape.

    "fmt"
    "io"
    "net/http"
    "regexp"
    "sort"
    "strings"
    "sync"
    "time"
)

// ScrapeResult holds everything we learned about one URL
type ScrapeResult struct {
    URL       string
    Title     string
    WordCount int
    Duration  time.Duration
    Err       error         // nil = success, non-nil = failure
}

// URLs to scrape — feel free to add your own
var targets = []string{
    "https://go.dev",
    "https://pkg.go.dev",
    "https://blog.golang.org",

    "https://this-does-not-exist.go",  // intentional failure
    "https://github.com/golang/go",
}
🧱
Err field in a struct — a common Go pattern

Embedding an error field directly in a result struct is how Go handles "this operation might fail" when you're collecting many results. Instead of panicking or stopping everything, each result carries its own error status. You inspect result.Err != nil at the end. Clean, explicit, Go.

Step 2 — The HTTP Client with Timeout

Always use a custom HTTP client with a timeout. The default Go HTTP client has no timeout — a single hanging URL could freeze your whole scraper forever:

Go main.go step 2 
// Package-level client — reused across all goroutines (it's safe to share)
var httpClient = &http.Client{
    Timeout: 10 * time.Second,
}

Step 3 — extractTitle() & countWords()

Two small helpers that work on raw HTML strings. We use a simple regex for the title tag, and strings.Fields for word counting:

Go main.go step 3 
// titleRegex matches the content of an HTML <title> tag
var titleRegex = regexp.MustCompile(`(?i)<title[^>]*>([^<]+)</title>`)

func extractTitle(html string) string {
    matches := titleRegex.FindStringSubmatch(html)
    if len(matches) < 2 {
        return "(no title)"
    }
    // Clean up whitespace and truncate long titles
    title := strings.TrimSpace(matches[1])
    if len(title) > 50 {
        title = title[:47] + "..."
    }
    return title
}

func countWords(html string) int {
    // Strip HTML tags with a simple regex, then count whitespace-delimited tokens
    tagRegex := regexp.MustCompile(`<[^>]+>`)
    text := tagRegex.ReplaceAllString(html, " ")
    return len(strings.Fields(text))
}

Step 4 — scrapeURL() — The Core Worker

This is the function each goroutine runs. It fetches one URL, records the duration, extracts the data, and sends a ScrapeResult into the channel:

Go scrapeURL — the goroutine worker, annotated
func scrapeURL(url string, results chan<- ScrapeResult, wg *sync.WaitGroup) {

defer wg.Done()
First line: defer Done(). Guarantees the WaitGroup counter decrements no matter how this function exits — success, error, or panic.

Start the clock. We want to know how long this URL took.

Make the HTTP request using our shared client. Returns a response and an error.
if err != nil {

Send a failure result — with the error and duration recorded. Don't return empty-handed.
return
Early return — Done() fires via defer, WaitGroup decrements, goroutine exits cleanly.
}
defer resp.Body.Close()
Always close the response body. Use defer so it closes when the function returns, even on error paths below.
body, err := io.ReadAll(resp.Body)
Read the full response body into memory as a byte slice.
if err != nil {
results <- ScrapeResult{URL: url, Err: err, Duration: time.Since(start)}
Send failure if body read fails.
return
}

Convert []byte to string for our text-based helpers.
results <- ScrapeResult{

Step 5 — printReport()

Sort results by duration and print a clean table. Successes get full details, failures show the error message:

Go main.go step 5 
func printReport(results []ScrapeResult, totalDuration time.Duration) {
    // Sort by duration — fastest first
    sort.Slice(results, func(i, j int) bool {
        return results[i].Duration < results[j].Duration
    })

    fmt.Println()
    success, failed := 0, 0

    for _, r := range results {
        if r.Err != nil {
            failed++
            fmt.Printf("  ❌ %4dms   %-45s %s\n",
                r.Duration.Milliseconds(),
                r.URL,
                r.Err.Error(),
            )
        } else {
            success++
            fmt.Printf("  ✅ %4dms   %-45s %-35q (%d words)\n",
                r.Duration.Milliseconds(),
                r.URL,
                r.Title,
                r.WordCount,
            )
        }
    }

    fmt.Printf("\n  %s\n", strings.Repeat("─", 50))
    fmt.Printf("  Scraped:  %d / %d URLs\n", success, len(results))
    fmt.Printf("  Failed:   %d / %d URLs\n", failed, len(results))
    fmt.Printf("  Total:    %dms\n", totalDuration.Milliseconds())
}

Step 6 — main() — The Orchestrator

The main function launches all goroutines, waits for them, collects results from the channel, and calls the report. This is the fan-out pattern from M08 in its final form:

Go main.go step 6 — wire it all up 
func main() {
    fmt.Println("🕷️  Gopher's Forge Web Scraper")
    fmt.Printf("Scraping %d URLs concurrently...\n", len(targets))

    // Buffered channel — one slot per URL
    results := make(chan ScrapeResult, len(targets))

    var wg sync.WaitGroup
    start := time.Now()

    // Fan-out: launch one goroutine per URL
    for _, url := range targets {
        wg.Add(1)
        go scrapeURL(url, results, &wg)
    }

    // Close channel once all goroutines finish
    // Run in its own goroutine so we don't block before collecting
    go func() {
        wg.Wait()
        close(results)
    }()

    // Collect all results from channel into a slice
    var allResults []ScrapeResult
    for r := range results {
        allResults = append(allResults, r)
    }

    totalDuration := time.Since(start)
    printReport(allResults, totalDuration)
}

§4 The Complete File

Gomain.go — complete
package main

import (
    "fmt"
    "io"
    "net/http"
    "regexp"
    "sort"
    "strings"
    "sync"
    "time"
)

type ScrapeResult struct {
    URL       string
    Title     string
    WordCount int
    Duration  time.Duration
    Err       error
}

var targets = []string{
    "https://go.dev",
    "https://pkg.go.dev",
    "https://blog.golang.org",
    "https://gobyexample.com",
    "https://this-does-not-exist.go",
    "https://github.com/golang/go",
}

var httpClient = &http.Client{Timeout: 10 * time.Second}

var titleRegex = regexp.MustCompile(`(?i)<title[^>]*>([^<]+)</title>`)

func extractTitle(html string) string {
    matches := titleRegex.FindStringSubmatch(html)
    if len(matches) < 2 { return "(no title)" }
    title := strings.TrimSpace(matches[1])
    if len(title) > 50 { title = title[:47] + "..." }
    return title
}

func countWords(html string) int {
    tagRegex := regexp.MustCompile(`<[^>]+>`)
    text := tagRegex.ReplaceAllString(html, " ")
    return len(strings.Fields(text))
}

func scrapeURL(url string, results chan<- ScrapeResult, wg *sync.WaitGroup) {
    defer wg.Done()
    start := time.Now()

    resp, err := httpClient.Get(url)
    if err != nil {
        results <- ScrapeResult{URL: url, Err: err, Duration: time.Since(start)}
        return
    }
    defer resp.Body.Close()

    body, err := io.ReadAll(resp.Body)
    if err != nil {
        results <- ScrapeResult{URL: url, Err: err, Duration: time.Since(start)}
        return
    }

    html := string(body)
    results <- ScrapeResult{
        URL:       url,
        Title:     extractTitle(html),
        WordCount: countWords(html),
        Duration:  time.Since(start),
        Err:       nil,
    }
}

func printReport(results []ScrapeResult, totalDuration time.Duration) {
    sort.Slice(results, func(i, j int) bool {
        return results[i].Duration < results[j].Duration
    })

    fmt.Println()
    success, failed := 0, 0

    for _, r := range results {
        if r.Err != nil {
            failed++
            fmt.Printf("  ❌ %4dms   %-45s %s\n",
                r.Duration.Milliseconds(), r.URL, r.Err.Error())
        } else {
            success++
            fmt.Printf("  ✅ %4dms   %-45s %-35q (%d words)\n",
                r.Duration.Milliseconds(), r.URL, r.Title, r.WordCount)
        }
    }

    fmt.Printf("\n  %s\n", strings.Repeat("─", 50))
    fmt.Printf("  Scraped:  %d / %d URLs\n", success, len(results))
    fmt.Printf("  Failed:   %d / %d URLs\n", failed, len(results))
    fmt.Printf("  Total:    %dms\n", totalDuration.Milliseconds())
}

func main() {
    fmt.Println("🕷️  Gopher's Forge Web Scraper")
    fmt.Printf("Scraping %d URLs concurrently...\n", len(targets))

    results := make(chan ScrapeResult, len(targets))
    var wg sync.WaitGroup
    start := time.Now()

    for _, url := range targets {
        wg.Add(1)
        go scrapeURL(url, results, &wg)
    }

    go func() {
        wg.Wait()
        close(results)
    }()

    var allResults []ScrapeResult
    for r := range results {
        allResults = append(allResults, r)
    }

    printReport(allResults, time.Since(start))
}

§6 Challenges — Push It Further

⚡ Challenge 1 — Rate Limiter

Add a rate limiter so you never fire more than N goroutines at the same time. Use a buffered channel as a semaphore — a classic Go pattern:

sem := make(chan struct{}, 3) — send before scraping, receive after. This limits concurrency to 3 parallel requests max, regardless of how many URLs you have.

⚡ Challenge 2 — Retry on Failure

Modify scrapeURL to retry failed requests up to 2 times with a 500ms pause between attempts. Only send a failure result if all retries are exhausted. Use time.Sleep and a loop.

🏆 Challenge 3 — Save to File

Write the results to a results.csv file using os.Create and fmt.Fprintf. Columns: URL, Title, WordCount, DurationMs, Status. Print "Saved to results.csv" at the end.

Hint: Use os.Create("results.csv"), write a header line, then loop over the sorted results writing one line per URL.

✅ Project #2 Checkpoint

  • You launched one goroutine per URL and ran them all in parallel
  • You used a send-only channel type chan<- to make intent explicit
  • You used defer wg.Done() as the very first line of each goroutine
  • You always defer resp.Body.Close() after a successful HTTP request
  • You embedded an error field in ScrapeResult for per-URL failure tracking
  • You closed the results channel from a dedicated goroutine after wg.Wait()
  • You ranged over a closed channel to collect all results into a slice
  • You used sort.Slice with an anonymous function comparator
  • One bad URL did not crash or hang the whole scraper — errors are isolated
